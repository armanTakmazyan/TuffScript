# Brainfuck Interpreter in TuffScript

A Brainfuck Interpreter written in TuffScript.

## Quick Start

1.  Run in the root directory:

    `yarn start`

2.  Input your Brainfuck code or try "Hello, World!":

    `++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.`
